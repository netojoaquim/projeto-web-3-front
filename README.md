# Projeto E-commerce

Este é um projeto de e-commerce com **frontend em React**.


## Como executar

1.  **Clone o repositório:**
    ```bash
    git clone <URL_DO_REPOSITORIO>
    cd <NOME_DO_REPOSITORIO>
    ```

2.  **Crie o arquivo `.env` para o frontend** 

    ```ini
    REACT_APP_BASE_URL=http://localhost:5000
    ```
3. **execute o projeto**
    ```ini
    npm run start
    ```



* **Frontend:** `http://localhost:3000`
